# HMCTS Task Manager Backend

## What is this?
Task management API for HMCTS caseworkers.

## How to run:
1. Install MySQL and create database: `taskmanager_db`
2. Update `application.properties` with your MySQL password
3. Run: `mvn spring-boot:run`
4. API runs at: http://localhost:8080

## Technologies:
- Java 17, Spring Boot, MySQL, Maven
